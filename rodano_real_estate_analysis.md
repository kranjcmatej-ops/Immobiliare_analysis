# Data Limitations

Some information cannot be fully isolated from the dataset:

-   Garden value (included in apartment price)
-   Exact commercial surface for each transaction
-   Exact price split between apartment and garage

Despite these limitations, the dataset provides a **reliable market
trend estimate**.
